Unit 2.1 Plan, select and use appropriate IT systems and software to meet needs

Originally the code editor I started with was CodePen as this was the software I was most familiar with, but it became rather quickly that CodePen was not the best tool for me to use while I have been making my website. Even though it does have three different sections for HTML, CSS and JavaScript, it’s a lot easier to make a mistake in comparison to Visual Code Studio. If you make a mistake it will tell you what the error is but will not give you any information on where the error is, which is a waste of a lot of time trying to figure especially if it's a small error such as missing one piece of syntax.

Visual Studio Code is a useful software editor one of the advantages that I personally found useful is whilst your writing code if it code into the editor if you make a mistake or spelling error or have written pieces of code that are undefined it will come up with a error sign under the code that is wrong this is very helpful as it saves time as instead of looking for a small mistake I was able to fix the problem immediately instead wasting time searching for errors. Another feature that Visual Studio Code has is similar to predictive text when you are writing your code it will give you a list of possible options to choose from and if you choose from that list the code will be written out for you this saves a lot of time and allows you to be more productive.

Trello is a very useful tool, using lists you can write down tasks and add multiple different tasks on different cards. This allows you to plan and actually see all of the different jobs that you need to complete. Once you have written a task down on one of the cards you can further expand upon the task showing the importance of the task by giving it a colour label, you can also add a checklist to the task so if a task has multiple steps instead of having multiple mini tasks and over cluttering your plan. You can also get to detail within the cards by using the description. This helps for example if you were in a group project that others would easily be able to follow the original person's intent when they thought about how the task should be completed. You can also set yourself deadlines for each task which allows you to prioritise between tasks depending on importance.
