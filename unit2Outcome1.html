Unit 1.6 Describing access issues that need taking into account - describe issues that people with disabilities may face when using webpages and how we can overcome them

The Web Accessibility Initiative describes requirements for Web access by people with physical, visual, hearing and cognitive or neurological disabilities. Many people who have disabilities need to use alternate websites that are more accessible to help use the web.

People with disabilities can use assistive technologies to help accomplish tasks that they could not complete on their own whilst dealing with the web. These tools use adaptive software, specially designed hardware devices or standard devices used in alternative ways to provide access for the web.

Some of examples of tools that help people with disabilities

Alternate keyboards or switches

Hardware or software devices, used by people with physical disabilities, that provide an alternate way of creating keystrokes that appear to come from the standard keyboard. Examples include on-screen keyboards, eyegaze keyboards, and sip-and-puff switches. Applications that can be operated entirely from the standard keyboard , with no mouse movements required, support single-switch access or access via alternative keyboards.

Screen magnifiers

Software used primarily by individuals with low vision that magnifies a portion of the screen for easier viewing. Note that at the same time screen magnifiers make presentations larger, they also reduce the area of the document that may be viewed. Some screen magnifiers therefore offer two views of the screen: one magnified and one default size for navigation.

Screen readers

Software used by individuals who are blind or have learning disabilities that interprets what is displayed on a screen, and directs it either to speech synthesis, for audio output, or refreshable braille, for tactile output. Some screen readers use the document tree (i.e., the parsed document) as their input. However, older screen readers make use of the rendered version of a document, meaning that document order or structure may be lost (e.g., when tables are used for layout) and their output may be confusing.

Braille Keyboards

A Braille keyboard is very different than a standard QWERTY keyboard. There are 8 keys which are used to compose the Braille letters. Of course, there are standard QWERTY keyboards which have Braille letters overlaid on them, but Braille keyboards are made in a way which makes navigation and locating the cursor much easier for visually-impaired people.

Refreshable Braille Displays

This is a type of Braille keyboard which also has output options. It contains small pins which can be raised to form the Braille letters. Displays range from 18 to 84 cells. A cursor allows the user to select which text to translate, and the pins are refreshed throughout the reading process. Braille displays are fairly expensive but are good options for people who use the computer a lot, need to speak and type at the same time, or who are deaf-blind and would not be able to use a Screen Reader.

Sound notification

Features of some operating systems that allow deaf or hard of hearing users to receive visual notification that a warning or error sound has been emitted by the computer.

Scanning software

Software used by individuals with some physical or cognitive disabilities that highlights or announces selection choices (e.g., menu items, groups of possible phrases, etc.) one at a time. A user selects a desired item by hitting a switch when the desired item is highlighted or announced.

Voice control

Voice recognition is used by people with some physical disabilities or temporary injuries to hands and forearms, as well as some users interested in greater convenience, and as an input method in some voice browsers. Applications that have full keyboard support can be used with voice recognition.[2]

Unit 1.7 Describe which are the appropriate filetypes for websites

Jpeg

For images one of the most commonly used file formats is a jpeg, a jpeg is a compressed data of an image, they are useful as you can save large images as relatively small files but there will usually be a loss in quality. The rate of compression varies depending on size of the file so there is a possibility to balance out the quality of the image.

Bmp

Bmp is a file format that stores raster images. These data files are a bitmap which is a map of colour values for each individual pixel within the image canvas. Bitmaps are used for high fidelity photographic images but because they store high quality images the size of the file makes them impractical for large images.

Gifs

A gif is another bitmap (raster image format. It is very popular with simple, non photographics images but it’s colour handling tends to break down with photos and can not handle gradients or colour depth very well. Gif files can be animated, allowing very short video loops to be embedded into otherwise static images.

Png

A PNG was created as an improved version of the GIF file. It is also a rasterized, or a bitmapped image file format. A PNG has a much broader colour palette and supports transparency, making it very popular for all sorts of graphic design and publishing work. PNG provides lossless compression, which aids in keeping file sizes down without a loss of quality.

Wav

The WAV file format is a (virtually) lossless, uncompressed audio format that preserves a high-fidelity digital reproduction of the sound waves as recorded by a microphone, input device, or digital synthesizer. It is most often used for raw sound samples and short in-app sound effects.

Mp3

Probably the most popular audio file format is the mp3, the mp3 uses lossy compression to lower file size at the expense of sound quality and fidelity. Given the audio equipment most often associated with this file format (headphones and computer speakers), there is not usually a noticeable drop in quality as compared to lossless formats.

Webm

WebM is an open video format, primarily conceived for use with relatively short video clips embedded on websites in the context of HTML5 Video. It is a relatively new file format, but it is supported and developed by Google and fills a need not being met by other formats, so its adoption across the web has been very fast. YouTube displays video in WebM format.

Mp4

MP4 is the successor to MPEG and is based on the QuickTime video format. Compared to the earlier MPEG formats, it has better compression, higher quality (less loss), and can carry more meta-data such as subtitles and chapter headings.
